# Agents Guide

This repository can be worked on effectively by coding agents by following a few core rules:

1. Keep changes minimal and focused on the issue being solved.
2. Run linting with `npm run lint`.
3. Run tests with `npm run test -- --no-coverage` for faster feedback.
4. Prefer existing patterns in `src/` over introducing new approaches.
5. Update docs when behavior or workflow changes.
